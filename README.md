### Ä°nstall

1. clone `git clone https://github.com/sivankanat/wppack.git .`

4. Update `build/build.js`

- Custom Post types
- Theme Options Panel
- Post Meta Boxes
- SCSS, ES Modules

### Usage

source dir `assets/src/`. uses sass and node modules

- for development mode use `npm start`
